Introduction
============

Simple tray application

Running and installing
======================

To run the applet, simply launch ghost.py: ./ghost.py.

Dependencies
=====================

* Twisted python - core + web - http://twistedmatrix.com/
* Autobahn python             - https://github.com/tavendo/AutobahnPython
* zmqproto                    - https://github.com/caedesvvv/zmqproto
* python-obelisk              - https://github.com/darkwallet/python-obelisk
